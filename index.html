Other options

'pageSelector'

The main page selector. Note: Don't set it to 'body'! Use any children of the body-tag. (Default: 'div.page')

'width'

The width of the off-canvas element. (Default: '100%')

'height'

The height of the off-canvas element. (Default: '100%')

'injectPosition'

Inject the off-canvas element 'before' or 'after' the page? (Default: 'before')

'pageClick'

Whether or not to hide the off-canvas element by click on the inactive page. (Boolean, Default: true)

Methods

'hide'

Hide current off-canvas element

'show'

Show current off-canvas element

'toggle'

Toggle current off-canvas element

'hideAll'

Hide all elements which are linked with the current pageSelector

Effects

This plug-in doesn't contain any kind of effects. All animations and effects are done with CSS3 transitions, 'cause jQuery animations can be laggy on mobile devices.

Required styles depending on the type and the position

Slide the off-canvas

Type: 'push'

.offcanvas-wrapper {
	transition: 1s;
}

Type: 'overlay'

.offcanvas-top {
	transition: top 1s;
}
.offcanvas-right {
	transition: right 1s;
}
.offcanvas-bottom {
	transition: bottom 1s;
}
.offcanvas-left {
	transition: left 1s;
}

Type: 'underlay'

.offcanvas-main {
	transition: top 1s, right 1s, bottom 1s, left 1s;
}

Type: 'Shrink' (comming soon)

-/-

Fade main page overlay

By default, the main page overlay is transparent. You can set e.g. background: rgba(0, 0, 0, .5); to .offcanvas-main:after, to set a light transparent overlay over the main page, if any off-canvas element is shown.

.offcanvas-main:after {
	transition: visibility 0s 1s, opacity 1s; /* Required for smooth fade-in-out */
}
.offcanvas-main.offcanvas-main-hidden:after {
	transition: opacity 1s;
}
