jQuery Offcanvas

Options

'mode'

The mode of the off-canvas effect.

Default: 'overlay'
Options: 'overlay', 'push', 'underlay', 'shrink' (comming soon)

'overlay' (default)

This is the off-canvas element
This is the main canvas

'push'

This is the off-canvas element
This is the main canvas

'underlay'

This is the off-canvas element
This is the main canvas

'shrink' (comming soon)

This is the off-canvas element
This is the main canvas

'position'

The position of the off-canvas element, relative to the main canvas.

Default: 'left'
Options: 'left', 'right', 'top', 'bottom'

'left' (default)

This is the off-canvas element
This is the main canvas

'right'

This is the off-canvas element
This is the main canvas

'top'

This is the off-canvas element
This is the main canvas

'bottom'

This is the off-canvas element
This is the main canvas

'mainCanvas'

The main canvas selector or jQuery object.
Note: Don't set it to 'body'! This plug-in wraps certain elements around the main canvas, so the main canvas could not be the body-tag. You can use a div, that wraps all the content of your page.

Default: '.main-canvas'
Options: jQuery selector or jQuery object

'width'

The width of the off-canvas element.

Default: '100%'
Options: <length>, <percentage>

'height'

The height of the off-canvas element.

Default: '100%'
Options: <length>, <percentage>

'injectPosition'

Inject position of the off-canvas element.

Default: 'before'
Options: 'before', 'after'

'easing'

Method of easing. It's recommend to include jQuery Transit to use CSS transitions. If you do not include this, jQuery Offcanvas will use jQuery's native .animate() function. To disable all animations, set 'animations' to false.

Note: If you use an other easing instead of 'linear' and CSS3 transitions are not supported in the browser, you should include a plugin, just like jQuery Easing Plugin or jQuery UI

Default: 'linear'
Options: 'linear', 'ease', 'in', 'out', 'in-out', 'snap', 'easeOutCubic', 'easeInOutCubic', 'easeInCirc', 'easeOutCirc', 'easeInOutCirc', 'easeInExpo', 'easeOutExpo', 'easeInOutExpo', 'easeInQuad', 'easeOutQuad', 'easeInOutQuad', 'easeInQuart', 'easeOutQuart', 'easeInOutQuart', 'easeInQuint', 'easeOutQuint', 'easeInOutQuint', 'easeInSine', 'easeOutSine', 'easeInOutSine', 'easeInack', 'easeOutBack', 'easeInOutBack'

'duration'

Duration of effects in ms. Set it to 0 (zero) to disable animations. (Doesn't include main canvas overlay)

Default: $.fx.speeds._default (Default jQuery speed: 400)
Options: integer

'animateFallback'

Enable/Disable fallback to native jQuery .animate().

Default: true
Options: boolean (true/false)

'mainCanvasOverlay'

Should there be an overlay over the main canvas, if any off-canvas element is active?

Default: true
Options: boolean (true/false)

'mainCanvasOverlayClick'

Should the main canvas overlay be click-able to hide current off-canvas element? (Requires 'mainCanvasOverlay' to be true)

Default: true
Options: boolean (true/false)

'mainCanvasOverlayDuration'

Duration of fade effect of the main canvas overlay in ms. Set it to 0 (zero) to disable this effect. (Requires 'mainCanvasOverlay' to be true)

Default: $.fx.speeds._default (Default jQuery speed: 400)
Options: integer

'cloneElement'

Clone the off-canvas element instead of moving it to the right position.

Default: true
Options: boolean (true/false)

Public methods

'hide'

Hide current off-canvas element.

Example: 
$('.offcanvas').hide();

'show'

Show current off-canvas element.

Example: 
$('.offcanvas').show();

'toggle'

Toggle current off-canvas element.

Example: 
$('.offcanvas').toggle();

'hideAll'

Hide all elements which are linked with the current main canvas.

Example: 
$('.offcanvas').hideAll();

Effects

This plug-in uses the jQuery Transit plug-in. All animations and effects are done with CSS3. If you don't want to include this plug-in, Offcanvas will use the normal jQuery $.fn.animate() function.
It is recommend to use the Transit plug-in, 'cause CSS3 animations are smoother (Especially on mobile devices)
